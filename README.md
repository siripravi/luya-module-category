# LUYA *Hierarchial data administration* Module

[![LUYA](https://img.shields.io/badge/Powered%20by-LUYA-brightgreen.svg)](https://luya.io)

*LUYA Nestedset module* based on [Yii2 Nestedset Extension](https://github.com/creocoder/yii2-nested-sets)

## Installation

For installation of this module, composer is required.

Install the extension through composer:

```sh
composer require siripravi/luya-module-category
```

Run the import command afterwards:

```sh
./luya import
```
./luya migrate
